## 🚀 Quick Start

Simply open `index.html` in a web browser or serve with any HTTP server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx http-server -p 8080
```

Then visit `http://localhost:8080` in your browser.
